# The Arts in Paris

Welcome to The Arts in Paris project! This repository contains the source code for a website that showcases a charming café located in the heart of Agadir's historic district, blending tradition with modern sophistication.

## Project Overview

The project is a website built using HTML, CSS, and Bootstrap. It features:
- A responsive design
- A video background on the home page
- Multiple sections such as Home, About Us, Menu, and Contact Us
- Google Maps integration
- Smooth scrolling and elegant transitions

## Visit the website live:
[restaurance.netlify.app](https://restaurance.netlify.app)


## Usage

- **Home Page**: Features a video background and an introduction to The Arts in Paris café.
- **About Us**: Learn about the history, atmosphere, and team of The Arts in Paris.
- **Menu**: Explore the delightful menu offerings.
- **Contact Us**: Contact information and Google Maps location.
